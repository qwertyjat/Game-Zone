# 🎮 GTA 6 Game Zone

Welcome to the official repository for **GTA 6 Game Zone**—a premium, next-generation luxury gaming lounge website architecture. Built with an elite dark aesthetic, vibrant neon accents, glassmorphic UI elements, and fluid animations.

---

## ✨ Features Engine
* **Luxury UI/UX:** Built using professional `Inter` and `Poppins` typography scales backed by high-end CSS neon glow variables.
* **Dynamic Battleground:** An interactive grid of 12 top-tier competitive titles populated on-the-fly via localized JavaScript arrays.
* **Fluid WhatsApp Booking Pipeline:** A fully functional booking form that automatically captures user selections (Platform, Time Slot, Duration) and encodes them securely into a mobile-ready WhatsApp API session (`wa.me`).
* **Performance Enhancements:** Smooth-scrolling custom anchors, animated numeric counters, interactive FAQ accordion architectures, and a responsive customer testimonial carousel.
* **Mobile-First Infrastructure:** 100% fluid layouts that scale down perfectly from massive 4K gaming monitors to mobile displays.

---

## 🛠️ Tech Stack & Protocols
* **Structure:** HTML5 (SEO optimized semantic structure)
* **Styling:** CSS3 (Advanced custom properties, Flexbox, Grid, Glassmorphic overlays)
* **Logic:** Vanilla JavaScript (No heavy frameworks, ensuring sub-1ms local processing speeds)
